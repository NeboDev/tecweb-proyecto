# tecweb-proyecto
## Dashboard de gestión de recursos  
Proyecto de la asignatura **Desarrollo de Aplicaciones Web - Otoño 2025**

---

## Descripción
Este proyecto consiste en un dashboard interactivo con funcionalidades desarrolladas en **jQuery** y **AJAX**, utilizando el microframework **Slim** como base para la API.

El sistema permite:
- Agregar recursos
- Gestionar recursos existentes
- Descargar recursos disponibles en la plataforma

---

## Tecnologías utilizadas
- PHP 8+
- Slim Framework
- MySQL / MariaDB
- jQuery
- AJAX
- HTML, CSS y JavaScript
